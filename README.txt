Face Recognition for Avtar & Human Face.
It uses wavelet transform for feature point detection(5 features per frame).
Multi SVM is used as classifier. 
Dataset used for Human faces from http://cswww.essex.ac.uk/mv/allfaces/faces94.zip
For Avtar images are captured from secondlife
Paper- Artificial Human Face Recognition via Daubechies Wavelet Transform and SVM
